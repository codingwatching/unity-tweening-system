# Supported Types

### Generics

When creating tweens using the generic functions, the following types of values can be animated:

- `float`
- `double`
- `long`
- `int`
- `Vector2`
- `Vector2Int`
- `Vector3`
- `Vector3Int`
- `Vector4`
- `Quaternion`
- `Rect`
- `Color`
